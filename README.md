# IMKit Web UI SDK

IMKit 提供在 Web 上建構即時通訊軟體所需的 1) JS SDK 和 2) UI SDK。

JS SDK 提供存取 IMKit Chat Server 的 API 和文件，透過 JS SDK 可進行即時通訊的各種操作，例如建立聊天室、加入聊天室、傳送訊息等。
UI SDK 提供在 Web 上建構即時通訊軟體所需的介面開發實作。
